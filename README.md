# Fitness Landscape Navigation

This repository contains code for generating and calculating ruggedness measures for various random fitness landscapes.

##  Overview

- Generate different classes of fitness landscapes
- Simulate adaptive walks under stochastic dynamics
- Identify local fitness peaks and their basins of attraction
- Compute ruggedness measures 
- Quantify accessibility of the global optimum

##  Implemented Models

- NK model: Tunable ruggedness via parameter \( K \)
- HoC (House-of-Cards): Completely uncorrelated random landscape
- RMF (Rough Mount Fuji): Additive slope c + noise
- Alternating (eggbox-like): Structured high/low fitness pattern
